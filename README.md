# Healthcare Cost Analysis Dashboard

This project demonstrates an end-to-end workflow for cleaning, analyzing, and visualizing healthcare cost and utilization data in Excel. The focus is on identifying major cost drivers, high-risk segments, and actionable insights for strategy and risk analysis.

## Features

- **Data Cleaning & Standardization:**  
  Removed inconsistencies and standardized categorical fields (Insurance Provider, Admission Type, Gender) using structured tables, Excel formulas, and data validation to ensure data integrity.

- **Derived Metrics:**  
  Calculated key metrics such as Age Group, Length of Stay, Cost per Day, and Cost Risk Band to segment patients and evaluate financial impact.

- **PivotTables & PivotCharts:**  
  Summarized costs by demographics, admission type, insurance provider, and risk band. Created charts to visualize trends and highlight high-cost segments.

- **Interactive Dashboard:**  
  Designed a cohesive Excel dashboard presenting KPIs, cost trends, and variance indicators for reporting and decision-making.

## Tools & Techniques

- Microsoft Excel: Tables, Formulas, PivotTables, PivotCharts  
- Data Cleaning & Validation  
- KPI Calculation & Segmentation  
- Dashboard Design & Visualization
